# Practice Exercises

## Beginner

1. Write a program to check whether a number is odd or even.
2. Write a program to find the largest among three numbers.
3. Print the multiplication table of a given number.
4. Calculate the sum of digits of a number.
5. Reverse a number using a loop.

## Intermediate

1. Write a function to check if a string is palindrome.
2. Find the second largest element in an array.
3. Count frequency of each word in a sentence using `map`.
4. Implement binary search using recursion.
5. Sort a vector of students by CGPA.

## OOP

1. Create a `Student` class with private data and public getter/setter methods.
2. Create a `Shape` base class and derive `Circle`, `Rectangle`, and `Triangle`.
3. Overload `+` for a custom `Time` class.
4. Build a small bank account management system.

## DSA

1. Implement linked list insertion at beginning and deletion by value.
2. Implement stack using linked list.
3. Implement queue using circular array.
4. Implement DFS and BFS for a graph.
5. Implement Dijkstra's shortest path algorithm.

## Advanced

1. Create a generic `Stack<T>` using class templates.
2. Use `unique_ptr` to manage a binary tree.
3. Create a thread-safe queue using `mutex` and `condition_variable`.
4. Implement Factory and Strategy design patterns.
5. Build a file-based student record system.
